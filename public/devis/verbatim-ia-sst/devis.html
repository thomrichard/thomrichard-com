Conditions et hypothèses

Ce devis est établi sous réserve des conditions suivantes :

  1. Le CRM client expose une API REST documentée permettant l'écriture dans la grille de prévention. Audit confirmatoire en début de mission.
  2. Le client fournit la liste des articles du CSTC à indexer. La récupération des contenus se fait depuis sources publiques (LégisQuébec, CNESST) sur la base de cette liste.
  3. La langue de la version 1 est le français québécois. L'anglais peut être ajouté en version 2 (hors devis).
  4. Les observations consignées ne contiennent pas de données nominatives sensibles.
  5. La validation humaine sur mobile précède le push CRM.
  6. Le mode hors ligne robuste n'est pas inclus dans cette v1. Il peut être proposé en option v1.5 après tests de la v1.

Si l'une de ces conditions ne se vérifie pas (notamment l'API CRM), un avenant est proposé après échange.
