Les MCP essentiels

10 connecteurs MCP pour brancher Claude à tes outils sans coder. Le complément du tuto complet Claude.

0.Avant de se lancer : natif vs MCP

Beaucoup d'outils sont déjà connectables nativement depuis Settings → Connectors de Claude. Pas besoin d'installer un MCP pour ceux-là.

Gmail · Google Calendar · Google Drive · GitHub · Notion · Linear · Asana · Stripe · HubSpot · Cloudflare

Les MCP ci-dessous, c'est tout ce qui n'est pas couvert nativement et qui change vraiment ton quotidien.

Comment installer un MCP en 2026 : Settings → Connectors → "Add MCP server". La plupart proposent un bouton "Install" en 1 clic. Pour les MCP self-hosted (n8n, Filesystem), suis le README du projet.

1.Data & CRM no-code

Airtable Officiel · Facile

LE stack data des solos. Lire, écrire, filtrer tes bases via Claude. Ton CRM, ton backlog éditorial, ton pipeline de prospection — tout devient pilotable en langage naturel.

Outil :clubsmart.s.gy/airtable MCP :airtable-mcp-server
Supabase Officiel · Moyen

Quand tu vas un cran plus loin que no-code. Postgres managé + auth + storage + edge functions. Le combo Claude + Supabase = MVP en quelques heures.

Outil :clubsmart.s.gy/supabase MCP :supabase-mcp

2.Automation no-code

n8n-mcp Community · Moyen

Génère, valide et déploie des workflows n8n directement depuis Claude. Tu décris ton automatisation en français, le MCP construit le workflow. Killer app pour les builders.

Outil :clubsmart.s.gy/n8n MCP :n8n-mcp
Zapier MCP Officiel · Facile

Si t'es plutôt Zapier que n8n. Déclencher des Zaps existants ou en créer via Claude. Plus de 7000 apps connectables d'un coup.

Outil :clubsmart.s.gy/zapier MCP :mcp.zapier.com
Make (via Zapier ou n8n) Indirect · Facile

Make n'a pas de MCP officiel à ce jour, mais tu peux le piloter via webhooks déclenchés depuis Claude. Astuce : passe par un MCP HTTP générique ou par n8n comme orchestrateur.

Outil :clubsmart.s.gy/make

3.Web & recherche

Firecrawl Officiel · Facile

Scrape n'importe quel site, même JS-rendered. Bien plus puissant que la web search native de Claude. Pour la veille, l'analyse concurrentielle, l'extraction structurée.

Outil :clubsmart.s.gy/firecrawl MCP :firecrawl-mcp
Brave Search Officiel · Facile

Alternative à la web search native sans tracking. Plus précis sur les recherches techniques et long tail. API gratuite jusqu'à 2000 requêtes / mois.

Outil :clubsmart.s.gy/brave MCP :brave-search-mcp
Apify Officiel · Moyen

Si tu as besoin de scrapers pré-construits (LinkedIn, Instagram, Twitter, Google Maps, TripAdvisor). Pour les pros de la prospection et de la veille structurée.

Outil :clubsmart.s.gy/apify MCP :actors-mcp-server

4.Création & contenu

Canva Officiel · Facile

Génération et édition de visuels en langage naturel. Pas pour le final, mais pour itérer vite sur des concepts de carrousels, vignettes, slides.

Outil :clubsmart.s.gy/canva MCP :canva.com/developers/mcp
Excalidraw Community · Facile

Diagrammes, schémas, croquis d'architecture. Sortie en fichiers .excalidraw éditables. Très pratique pour les scripts vidéo, les briefs clients, les explications visuelles.

Outil :clubsmart.s.gy/excalidraw MCP :mcp_excalidraw
VidIQ Officiel · Facile

Pour les créateurs YouTube. Stats chaîne, outliers, keyword research, scoring de titres et de miniatures — le tout depuis Claude. Compagnon idéal du skill youtube-packaging.

Outil :clubsmart.s.gy/vidiq MCP :vidiq.com/mcp

5.Mon stack perso (Thomas, Club SMART)

Pour donner un exemple concret, voilà les MCP que j'ai branchés sur mon Claude au quotidien :

Airtable Mon CRM Club SMART + mon backlog éditorial + ma base concepts
n8n-mcp Génération de workflows pour mes clients freelance et mes pipelines internes
VidIQ Veille créateurs + scoring de titres et miniatures avant publication
Excalidraw Diagrammes pour scripts YouTube et briefs clients
Firecrawl Veille concurrentielle et extraction de pages produits
Pas la peine de tout installer. Commence par 1-2 MCP qui correspondent à tes outils du quotidien. Ajoute le reste au fur et à mesure que tu sens le besoin.
